chrome://flags/

  • overlay-scrollbars : 使用自动收起的滚动条，不占用额外空间(全平台)
  • enable-smooth-scrolling : 像Mac和手机一样平滑滚动(仅Linux可用)
  • enable-download-resumption : 恢复下载
