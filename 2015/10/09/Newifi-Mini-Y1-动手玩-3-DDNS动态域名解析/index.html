Newifi Mini(Y1) 动手玩(3) - DDNS动态域名解析

发表于 2015-10-09 | |

安装 ddns

Openwrt系统应该是预装了ddns，小云OS没有预装ddns，我们可以通过opkg装上ddns

  • 配置 opkg
    可参考 Newifi Mini(Y1) 动手玩(2) - opkg

  • 在管理页面搜索 ddns

  • 安装 ddns-scripts 和 luci-app-ddns

安装完成即可在 服务中找到动态DNS页面

启动 ddns

系统 -> 启动项 -> 启动ddns

配置 ddns

  • 我们需要选择一个 ddns 服务商，这里推荐 3322.org (http://www.pubyun.com/ )，注册后创建一个动态域名。
  • 后台配置
  • 重启路由器

自定义域名

使用 3322 的二级域名当然不能满足我们的要求，要如何实现自定义域名呢？

  • 注册域名
  • 在你的DNS服务商添加一个CNAME映射到动态域名中

常见问题

为什么完成上面步骤还是不能使用自定义域名访问呢？

  1. 检查防火墙设置
  2. 检查IP端口是否被关闭
    使用 端口扫描 查看80端口是否被关闭，如果关闭，我们可以换个端口
修改保存后，重启uhttpd服务

安全问题

路由器密码不要使用弱密码，如果路由器被黑客入侵，密码安全就没有保障了。ssh应该关闭密码登录使用密钥登录。
