Android - 单元测试之 UI Testing

Android Studio不知从什么版本开始, 新建项目后就会默认生成androidTest目录, 可以看出Google在推广Android单元测试. 编写高品质的App, 测试代码是必不可少的.

Android的测试框架分为:

  • Espresso : UI 测试框架 - 适用于 界面功能 测试
  • UI Automator : UI 测试框架 - 适用于 跨app界面功能 测试
  • AndroidJUnitRunner : 兼容Android版本的Junit 4

其中Espresso是最新的UI Test框架, 支持UI的创建测试和用户流程测试, api非常简明, 运行也非常可靠. 也是很适合来写白盒测试, 可以实现对代码的细节测试.
