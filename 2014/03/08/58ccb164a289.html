# 面试经验

应届生社招面经

珠海 1.1k 3 min

背景

  1. 东软双选会来的企业，基本上都没有移动端的岗位；
  2. 本想去川大蹭校招的，来回太远了，青城山区的学子求职太恼火了；

所以错过了校招，这个时间点面试都是直接招聘网站社招，当然绝大多数都是成都的小型创业公司，大牛看到勿喷，本文仅仅作为我的一点点经验

一、准备工作

简历

  1. 网投平台一般有简历模板，这个要尽量填满，填完之后预览，看哪里要修改的，修改需要突出显示的。(比如说我就把我的博客信息，直接填在专业描述里面，因为这 样可以显示在最前面)
  2. 邮件投简历，需要你自己做一个简历文档，找模板，注意最后最好导出成 PDF 格式，不要用 Doc
  3. 通知面试最好备一份纸质简历，有些公司可能不会把你发给他的简历打印出来

笔试

  1. 临时抱佛脚吧，直接上网找笔试题，然后做，一般找个两三套就够了
  2. Java 主要知识点:线程同步，异常处理，集合(Collection 的那些子类，Arraylist 和 LinkedList 的不同)，多态性，final, finally, finalize 的不同

面试

  1. 有笔试，可能会对你笔试内容做个询问
  2. 会让你介绍你做过的项目
  3. 基本上所有公司都有问 Android Listview 的优化，Bitmap 的处理，网络异步通讯

二、面试经历

成都游戏工厂科技有限公司

  1. 朋友内推
  2. 然后笔试，由于我的主语言的 Java，他们就给的全 Java 的题目，都还比较基础。涉及到二叉树遍历、图论算最短距离、异常处理、排序、查找等等
  3. 然后让回家等面试结果，一周过后他们的电话我没有接到，十多个未接来电，应该是叫过去面试的，但是我没有回电，因为第二天要去魅族面试

成都梦启科技有限公司

  1. 51Job
  2. 面试围绕我的博客+项目，主要聊技术，也蛮愉快的
  3. 三天后通知我可以去上班了。但是半年实习期，还有半年试用期(90%薪水)。老子只有三个月就毕业了，你给我半年的实习期

成都亿铂科技有限公司

  1. 51Job
  2. 面试围绕我的博客+项目，他们马上要启动的项目和我的某个项目有点类似，所以找到了我。聊了聊技术，也蛮愉快的
  3. CEO 是川大毕业的，四川人，当场就说薪水方面没有什么问题，转正问题毕业后细谈，要我第二天就来上班。（当时是明天还有个面试，我说周五给他答复，然后就离开了）

成都铁皮人科技有限公司

  1. 51Job + 朋友内推
  2. 三个创始人都是重庆人，面试也重庆话整起，所以沟通非常愉快
  3. 面试内容基本上围绕他们马上要做的项目展开问(Socket 编程，JNI，Android 适配)，技术问完就问待遇，然后拉拉家常，兴趣爱好(基本上到这步十拿九稳了)
  4. 铁皮人是我见过最有良心的公司，应届生未毕业，都是直接签劳动合同，良心企业有木有，给个赞

魅族科技有限公司

  1. 这个最简单粗暴，直接给李楠发了封 300 多字的邮件(本人中魅毒太深)
  2. 面试时间最长，前后弄了半个月，发了邮件等了三天，人力部问我什么时候有时间电话面试，然后约了周一电话面试，然后同周周四 HR 通知我什么时候有时间来公司现场面试，然后去魅族笔试+面试
  3. 最后成功在魅族实习

其他几家创业型公司

  1. 51Job
  2. 面试基本上是拿我博客+项目来聊，也主要是 Listview 性能优化，开源社区，线程，异步，自定义控件，动画等等。

三、总结

  1. 通过笔试题查漏补缺，不会的自己去写示例代码，一两个晚上就能搞定全部内容，临时抱抱佛脚，完全行的
  2. 然后平时多做项目，多写博客，这个可以看出你的学习能力和学习态度，同时也是面试的谈资
  3. 发动的你人脉资源，各种内推资源搞起
