2014 The End

【2014的匆匆而过】

这一年毕业了，这一年工作了；
这一年去了珠海，去了深圳，去了澳门，去了北京；
这一年去了魅族，做了职业码农，码畜，码XX。

一直想找一份Remote Working的工作，也在为此而努力，对目前的状态不是很满意，全在加班，没有时间学习新的知识，很怕这种没有进步的日子，七月份入职，前三个月九九六的加班，后三个月没有九九六，但也是一周六天班。

【2015的学习目标】

  • 英语 (Remote Working 很重要的一点)
  • 深入Android 动画 自定义控件
  • 学习下OpenGL
  • 空闲时间写写iOS或者Cocos2d
  • 空闲时间学学QT

【2015的独立项目】

  • Formater (一款MP3封面管理软件)
  • Listen (一款优雅的本地音乐播放器)

2014是幸运的一年，去了一直很想去魅族科技，感谢老大对我的赏识，因为魅族也认识了一群很牛逼的人，魅族是一家说大不大说小也不小的公司，公司成立十年，创业却刚刚开始，希望能趁着创业之际，好好充实自己，实现Remote Working的梦想。

Bye，年终总结，明年再见…
