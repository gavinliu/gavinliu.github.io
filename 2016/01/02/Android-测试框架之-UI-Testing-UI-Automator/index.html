Android - 测试框架之 UI Testing - UiAutomator

发表于 2016-01-02   |   分类于 Android Notes   |     |  

UiAutomator和Espresso最大的不同是支持多个App，所以UiAutomator是所有Android测试工程师的必备技能。

UiAutomator2.0版本依赖了Instrumentation使得UiAutomator 2可以轻松的拿到Activity的Context，就可以像写Android应用程序一样，实现很多功能。
